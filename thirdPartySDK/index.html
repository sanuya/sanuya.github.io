撒吖接入第三方SDK明细及权限说明

撒吖接入第三方SDK明细及权限说明

 

为了适配在不同手机设备上使用，保障撒吖App相关功能的实现与安全稳定的运行，撒吖接入了第三方提供的SDK。

撒吖App所接入的SDK可能会调用您的设备权限、获取您的相关信息；撒吖App会对合作方获取信息的SDK进行严格的安全监测，以保障您的信息安全；同时尽可能及时更新并向您说明撒吖App所接入第三方SDK的最新情况。

撒吖App接入的相关第三方SDK目录如下：

1、EventBus SDK

功能/服务：用户开发者Android事件发布/订阅框架

信息类型：唯一设备识别码（安卓 ID）

调用权限：无

官网链接：https://github.com/greenrobot/EventBus

 

2、ExoPlayer SDK

功能/服务：应用层媒体播放器开源框架

信息类型：无

调用权限：无

官网链接：https://github.com/google/ExoPlayer

 

3、Glide SDK

功能/服务：一个快速高效的Android图片加载的开源框架

信息类型：无

调用权限：无

官网链接：https://github.com/bumptech/glide

 

4、Okhttp3 SDK

功能/服务：用于Android中请求网络的开源框架

信息类型：无

调用权限：无

官网链接：https://square.github.io/okhttp/

 

5、Okio SDK

功能/服务：用于Android中IO操作的开源框架

信息类型：无

调用权限：无

官网链接：https://github.com/square/okio

 

6、RxJava SDK

功能/服务：用于Android中线程调度的开源框架

信息类型：无

调用权限：无

官网链接：https://github.com/ReactiveX/RxJava

 

7、SimpleCropView SDK

功能/服务：用于Android中圆角控件的开源框架

信息类型：无

调用权限：无

官网链接：https://github.com/igreenwood/SimpleCropView

 

8、数美SDK

运营公司：北京数美时代科技有限公司

功能/服务：用于风险控制和反欺诈，以保障账户和交易安全

信息类型：设备 MAC 地址（后台获取）、蓝牙 MAC 地址、SIM 卡信息（ICCID、IMSI、运营商信息）、唯一设备识别码（IMEI[后台获取]、IDFA、安卓 ID 、设备序列号[后台获取]）、设备的基本配置信息（如CPU、内存、屏幕、型号、操作系统）、网络信息（如 WIFI 及蜂窝网络信息、IP 地址、时间戳）、安装软件列表、系统及软件相关配置信息

调用权限：无

官网链接：https://www.ishumei.com/

隐私政策链接：https://www.ishumei.com/legal/cn/privacy.html

 

9、统计SDK

运营公司：神策网络科技(北京)有限公司

功能/服务：数据统计及分析

信息类型：唯一设备识别码（安卓 ID、设备序列号）、读取已安装包名列表

调用权限：无

官网链接：https://www.sensorsdata.cn/

隐私政策链接：https://www.sensorsdata.cn/compliance/privacy.html

 

10、热云SDK

运营公司：北京热云科技有限公司

功能/服务：用于广告数据分析服务

信息类型： 唯一设备识别码（安卓 ID）、IMEI，MAC地址、MEID、OAID、IP地址、当前运行应用进程 、设备运营商信息、设备型号、终端制造厂商、终端设备操作系统版本、IDFA&IDFV（若您使用的系统版本为IOS 14.5及以上的，未经您同意，第三方SDK将不会收集该信息）

调用权限：无

官网链接：https://new.trackingio.com/#/home

隐私政策链接：https://new.trackingio.com/#/policy

 

11、声网SDK

运营公司：上海兆言网络科技有限公司

功能/服务：获取使用用户麦克风功能，用于直播间上麦语音聊天     

信息类型：音频信息（语音信息）

调用权限：麦克风

官网链接：https://www.agora.io/cn/

隐私政策链接：https://www.agora.io/cn/privacy-policy
